Project ini berfokus pada optimasi kampanye deposito bank menggunakan machine learning untuk 
meningkatkan efektivitas targeting pelanggan. Dengan memanfaatkan data marketing perbankan, project ini 
bertujuan mengidentifikasi nasabah yang paling berpotensi melakukan deposit sehingga campaign dapat 
dijalankan lebih efisien, lebih tepat sasaran, dan menghasilkan ROI yang lebih tinggi. 
Keywords: Machine Learning, Predictive Modeling, Marketing Analytics, Campaign Optimization, XGBoost
